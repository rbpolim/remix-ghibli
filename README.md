# Project

A website to Explore & Search about all films by Studio Ghibli, built w/ [Remix](https://remix.run/) & Tailwind.

## Tech Stack

**Client:** Remix, TailwindCSS

**Server:** Studio Ghibli API

## Techs

- Remix
- TailWindCSS
- [Studio Ghibli API](https://ghibliapi.herokuapp.com/)
